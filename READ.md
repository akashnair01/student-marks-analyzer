# Student Marks Analyzer

This project analyzes student marks using Python.

Features:
- Show Average marks 
- Show higher marks 
- Show average marks
- Total marks
- Display graph(line,bar)
Tools Used:
- Python
- Pandas
- Matplotlib

## Files
- students.csv
- student_analysis.py

## Requirements
Install libraries:

pip install pandas
pip install matplotlib

## Run the Program

Open terminal and run:

python student_analysis.py
